## Exporting data

```sh
{{{apiUrl}}}/v2/projects/export?ak={{{apiKey}}}
```

or with curl

```shell-session
curl "{{{apiUrl}}}/v2/projects/export?ak={{{apiKey}}}" \
--output data.zip
unzip data.zip
rm data.zip
```
